# generated/

Este diretório é populado automaticamente pelo Astro durante o build ou start do servidor de desenvolvimento.

Arquivos gerados:
- `news.json` — Notícias buscadas dos feeds RSS em tempo de build
- `tax-data.json` — Dados da reforma tributária (local ou remoto)

Estes arquivos estão no .gitignore. Na primeira execução, o `data-fetcher` os cria automaticamente.
